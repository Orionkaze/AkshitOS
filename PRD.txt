1. PRODUCT OVERVIEW

AkshitOS is a SaaS-style, authenticated productivity dashboard designed for developers and engineering students.

It integrates multiple self-optimization tools into a single modular system with secure authentication and database-backed persistence.

The system is:

Authenticated (Email + Google OAuth)

Single-user role (no admin system)

Database-backed (PostgreSQL via Prisma)

Modular

Production-deployable

This is not a demo app.
This is a structured SaaS foundation.

6. API ARCHITECTURE

Each module must have RESTful endpoints.

Pattern:

GET /api/{module}
POST /api/{module}
PUT /api/{module}/{id}
DELETE /api/{module}/{id}

All endpoints must:

Validate session

Extract userId from session

Scope queries by userId

Validate input

Return structured JSON responses

Handle errors with proper status codes
